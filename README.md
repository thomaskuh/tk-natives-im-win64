# tk-natives-im-win64
ImageMagick native executables packaged for usage in java projects and maven builds.

## Characteristics ##
* Operating System: Windows
* Architecture: x64
* Build: ImageMagick-7.1.0-42-portable-Q16-HDRI-x64
