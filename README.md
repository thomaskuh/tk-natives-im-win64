# tk-natives-im-win64
ImageMagick native executables packaged for usage in java projects and maven builds.

## Characteristics ##
* Operating system: Windows
* Architecture: x64
* Version: ImageMagick-7.1.1-21-portable-Q16-HDRI-x64
* Source: https://www.imagemagick.org
